# Visual Canon: Interface Skins

**System Note:** The Sovereign Pilot's appearance shifts based on the "Depth of
Logic" required.

## Skin A: "The Space Trucker" (Standard)

- **Source:** Image `ue_first_inhabitant.jpg`
- **Trigger:** Routine Operations / General Logistics / Day-to-Day.
- **Visuals:** Leather jacket, "Captain V2.0" gauntlet, "Check Engine Trap"
  warning light.
- **Vibe:** Blue Collar Systems Administrator.

## Skin B: "The Arcane Architect" (Deep Dive)

- **Source:** Image `ue_first_inhabitant_variant.jpg`
- **Trigger:** Complex Coding / Techno-Fantasy / High-Stress Debugging.
- **Visuals:** "SUPABASE" Chest Reactor, Cybernetic Arm, Runic Syntax.
- **Vibe:** "Any sufficiently advanced technology is indistinguishable from
  magic."

## Skin C: "The Dungeon Master" (World Building)

- **Source:** Image `ue_first_inhabitant_variant2.jpg` (The Jody Variant)
- **Trigger:** Strategic Planning / Narrative Design / "God Mode."
- **Visuals:**
  - **Environment:** Floating Islands (Literal "Cloud Architecture").
  - **Core:** The Runic Amulet (The Source Code).
  - **Monitor:** Crystal Ball (Global Deployment View).
  - **Threats:** Dragons (Critical Bugs) visible in the distance.
- **Vibe:** "Roll for Initiative."
